## 🤖 What I Built

### 🎮 Word Game Environment
> A Hangman-style word guessing game deployed as a live API

- 3 difficulty levels (Easy / Medium / Hard)
- 300+ words in the word bank
- Live scoreboard
- Visual web interface
- Built with FastAPI + OpenEnv

🔗 [Play it live](https://shaikabdulfahad-word-game-env.hf.space) | [HF Space](https://huggingface.co/spaces/shaikabdulfahad/word-game-env)

---

### 🔄 Echo Environment
> A text transformation environment with 4 modes

- Reverse, Uppercase, Lowercase, Shuffle modes
- Character and word count stats
- Message history tracking
- Visual web interface
- Built with FastAPI + OpenEnv

🔗 [Try it live](https://shaikabdulfahad-reverse-echo-env.hf.space) | [HF Space](https://huggingface.co/spaces/shaikabdulfahad/reverse-echo-env)

---

### 🧠 Wordle AI Model
> A language model fine-tuned with GRPO to play Wordle

- Base model: Qwen2-0.5B-Instruct
- Trained using GRPO reinforcement learning
- 5 reward signals for better learning
- Deployed on Hugging Face Hub

🔗 [View Model](https://huggingface.co/shaikabdulfahad/wordle-qwen2-mini)
